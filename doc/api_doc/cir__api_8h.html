void *  irq_rx_end_arg 
)

Set the protocol and tx parameters.

Parameters
objCIR object define in application software.
prx_protocolSet the rx protocol table.
output_initSet the initial output.
  • 0 : The output is low.
  • 1 : The output is high.
carrierSet the waveform with carrier or no carrier.
  • 0 : The carrier symbol has no carrier.
  • 1 : The carrier symbol has carrier.
rx_sclkdivThe divisor of the system clock to generate a proper clock for the input sampling and trigger tick event.
  • 0 : Set the appropriate value automatically. (Suggestion)
  • 1 ~ 255 : SCLK / rx_sclkdiv.
symbol_time_tolerance_usSet the time tolerance for demodulating RX symbols. This unit is us.
carrier_tolerance_cntSet the tolerance of carrier numbers for demodulating RX symbols.
  • 0 : This setting is "0" when this waveform is no carrier.
check_frame_end_time_usSet the end time after receiving the last symbol. This unit is us.
irq_tx_end_cbUser defined IRQ callback function. When finish to receive data, generate the interrupt.
irq_tx_end_argUser defined IRQ callback parameter.
Return values
none
