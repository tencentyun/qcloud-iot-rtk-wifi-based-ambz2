u8  direction,

)

To setup the PWM duty auto adjustment interrupt.

Parameters
objPWM object define in application software.
callbackThe callback function. It will be called when the interrupt is accurred.
directionThe bit map to enable/disable the interrupt. Bit 1 control the interrupt of duty duration reachs the up limit. Bit 0 control the interrupt of duty duration reachs the down limit.
enableTo enable(1) or disable(0) the interrupt. For interrupt disable, the arguments 'callback' & 'arg' are ignored.
Returns
