int gpio_irq_debounce_set ( gpio_irq_t *  obj,
uint32_t  debounce_us,
u8  enable 
)

To enables or disable the debounce function of the given GPIO IRQ pin. The debounce resource(circuit) is limited, not all GPIO pin can has debounce function.

Parameters
[in]pgpio_irq_adapterThe GPIO IRQ pin adapter.
[in]debounce_usThe time filter for the debounce, in micro-second. But the time resolution is 31.25us (1/32K) and the maximum time is 512 ms.
[in]enablethis parameter can be one of the following values:
  • 0 disable gpio debounce interrupt
  • 1 enable gpio debounce interrupt
Returns
0: Setting Succeed.
-1: Setting Fail.
