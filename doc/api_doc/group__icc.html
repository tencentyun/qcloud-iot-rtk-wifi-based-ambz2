Sends an ICC command to the LS platform.

Parameters
[in]cmdThe ICC command word 0, it contains the command type and the data word0.
[in]cmd_para1The ICC command parameter(data word1).
[in]timeout_usThe period, in micro-second, to wait the completion of command sending. -Value 0: no wait. -Value 0xFFFFFFFF: wait forever.
[in]task_yieldThe OS task yield function. The ICC HAL will call this function while wait for the command sending is finished.

Sends a ICC message (with message data) to peer.

Parameters
[in]msg_typeThe message type of the ICC message to be send.
[in]frame_typeThe frame type(category) of the message type.
[in]q_idThe message queue ID. It assign the message queue to send this message(with its data).
[in]msg_dataThe buffer of the message data to be send.
[in]msg_lenThe length(in byte) of data in the buffer msg_data to be send.
[in]callbackThe call-back function for the message data transfer is done.
[in]callback_argThe argument of the call-back function.
Returns
0: Message send OK. (the TX request adds to the TX pending queue OK).
< 0: Gor error on Message sending.
